span { border: 2px solid red; font-size: 5rem; } p { border: 2px solid blue; font-size: 5rem; width: auto; height: auto; padding: auto; margin-inline: auto; } img { border: 2px solid green; padding: 20px; margin-inline: 20px; } .imagem2 { margin-bottom: 2rem; } /* display: inline (span, strong, em, a, label, li) - faz com que o elemento se comporte como palavra - ocupa somente o espaço necessario para o conteudo - não épossivel utilizar width e height - padding e margin funcionam apenas na horizontal (parcialmente) display: block (header, main, div, sectioon, p, figure, footer, ul, ol, table) - faz com que o elemento se comporte como uma caixa - ocupa toda a largura disponivel. - é possivel utilizar o width, height, padding e margin perfeitamente display: inline-block (button, im, input) - mistura os comportamentos do display inline e do display block - elemento se comporte como uma palavra - elemento se comporte como uma caixa - ocupa de largura apenas o espaço necessario para o conteudo - é possivel utilizar o width, height, padding e margin perfeitamente */
